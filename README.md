# Anime Dating Server

This is a REST API for [DatingSwiftUI](https://github.com/armanabkar/DatingSwiftUI) iOS application. Built with Flask.

```bash
# Create virtual env
python3 -m venv venv 

# Activate virualenv:
. venv/bin/activate

# Install requirements and run flask 
pip install -r requirements.txt
flask run
```
